# Эмиль Хаматнуров — визитка-дашборд

Интерактивная одностраничная визитка frontend-разработчика, оформленная как «приборная панель»: ключевые показатели, радиальные индикаторы владения стеком, бортовой журнал опыта и контакты.

Концепция приборной панели не случайна — она отсылает к тому, что я делаю (WebTools — средство визуализации параметров баз данных) и к авиационному техническому университету.

## Стек

- **React 18 + TypeScript** — компонентная архитектура
- **Vite** — сборка и dev-сервер
- **SCSS Modules** — изолированные стили + дизайн-токены (`src/styles/tokens.scss`)
- Анимации на CSS + `IntersectionObserver` (без тяжёлых зависимостей)
- Шрифты: Unbounded (акцент), Golos Text (текст), JetBrains Mono (данные)

## Запуск

```bash
npm install
npm run dev      # http://localhost:5173
```

## Сборка

```bash
npm run build    # типобезопасная сборка в dist/
npm run preview  # предпросмотр прод-сборки
```

## Где править контент

Весь текст — стек, опыт, проекты, образование, контакты — в одном файле:
`src/data/resume.ts`. Меняешь данные — обновляется вся страница.

## Структура

```
src/
├─ data/resume.ts        # единый источник данных
├─ components/
│  ├─ Hero.tsx           # шапка + кластер показателей
│  ├─ SkillGauge.tsx     # радиальный индикатор навыка
│  ├─ SideNav.tsx        # навигация по разделам (scroll-spy)
│  └─ ...
├─ styles/
│  ├─ tokens.scss        # палитра, типографика, миксины
│  └─ global.scss        # базовые стили
└─ App.tsx               # сборка секций
```
